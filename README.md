# CryptoSphere: Real-Time Data and Predictions

A comprehensive cryptocurrency analysis and price prediction platform built as a Final Year Project at Guru Nanak Dev University (2020-2024).

## 📄 Project Report
Full project report available at: https://github.com/SHRIYA206/CryptoSphere/blob/main/CryptoSphere_FinalYear_Project_Report_ShriyaArora.pdf

## Overview
CryptoSphere integrates deep learning-based price forecasting with real-time market data, live news, interactive visualizations, and an AI-powered chatbot to help users make informed cryptocurrency investment decisions.

## Features
- Price prediction using Simple RNN (TensorFlow/Keras)
- Real-time cryptocurrency data via LiveCoinWatch API
- Interactive visualizations (OHLC, MACD, Scatter plots)
- Live crypto news aggregator
- AI-powered CryptoChatbot
- User authentication system

## Tech Stack
Python, Streamlit, TensorFlow, Keras, Pandas, NumPy, Plotly, Yahoo Finance API

## Note
Some third-party APIs used in this project may have changed or been deprecated since original development in 2024. The core RNN prediction model and visualization components remain functional.
